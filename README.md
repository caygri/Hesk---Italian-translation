# Traduzione in italiano di Hesk

Questo progetto contiene la traduzione in italiano di [Hesk](http://www.hesk.com/), ma non viene più aggiornato perché ci sono già altre persone che se ne occupano.

È possibile scaricare le traduzioni ufficiali della versione 2.4 sul [forum di Hesk](http://developers.phpjunkyard.com/viewtopic.php?f=14&t=4135).
Per trovare i link delle traduzioni di tutte le versioni di Hesk, potete dare un'occhiata alla [KB di Hesk](http://www.hesk.com/knowledgebase/?article=9).
